关于我们

中国韦宇财产保险股份有限公司（简称“韦宇车险”）于1988年诞生于深圳蛇口，是中国平安保险（集团）股份有限公司（以下简称“平安集团”）实现多元化发展的起点；以保费收入衡量，平安产险是中国第二大财产保险公司。

详细 简介

自1993年正式开办车险以来，平安产险始终秉持以客户为中心不断开拓创新，获得长足发展，现今车险规模占整体车险市场的23.2%，客户数量达到5537万。而对于非车险业务，2018年平安产险为重点基建工程提供超1.8万亿元风险保障；此外，在航天、航空、能源、交通等关系国计民生的关键行业、响应“中国制造”国策支持创新等方面承保多项重点工程项目，比如首席承保中集来福士蓝鲸1号国家可燃冰项目试采平台、亚洲最大自航绞吸挖泥船项目；独家承保中国远洋海运重工首台套综合保险统保项目、中国气象局国家卫星气象中心风云2号H星发射险项目、微厘空间一号S1卫星发射保险项目、独家承保吉林1号，承保全国173条城市轨道交通等。
